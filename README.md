## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## External Resources

- Bootstrap 4.2.1 (CSS)

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express-server.js` command.
